## ğŸ’¡ Tips & Trik

### 1. **Menulis Paragraf yang Baik**

**âŒ Jangan:**
```

bencana di wilayah Jakarta Timur pada tanggal 15 Januari 2026 dengan 
bantuan berupa paket sembako dan kebutuhan pokok lainnya yang diserahkan 
langsung oleh tim relawan kami yang telah dilatih khusus...
```

**âœ… Lakukan:**
```

Bantuan berupa paket sembako dan kebutuhan pokok lainnya. 
Program ini dilaksanakan pada tanggal 15 Januari 2026 di wilayah Jakarta Timur.
```

### 2. **Menggunakan Heading**

**Struktur Heading yang Benar:**
```
# Judul Artikel (Auto-generated, jangan tulis manual)

## Topik Utama 1
Konten...

### Sub-topik 1.1
Konten...

### Sub-topik 1.2
Konten...

## Topik Utama 2
Konten...
```

**âŒ Jangan Skip Level:**
```
## Topik Utama
#### Sub-topik (Skip H3)
```

### 3. **Bullet Points vs Paragraf**

**Gunakan Bullet Points Untuk:**
- âœ… Lokasi kegiatan
- âœ… Tanggal dan waktu
- âœ… Jumlah bantuan
- âœ… Jenis program
- âœ… Data statistik

**Gunakan Paragraf Untuk:**
- âœ… Penjelasan konteks
- âœ… Cerita atau narasi
- âœ… Kesimpulan
- âœ… Ajakan atau call-to-action

### 4. **Menambahkan Gambar**

1. Klik tombol **"Upload Image"** di toolbar editor
2. Pilih gambar dari komputer (max 2MB)
3. Gambar akan otomatis ter-upload ke GitHub
4. Gambar akan mendapat styling otomatis:
   - Frame tipis 4px
   - Shadow halus
   - Rounded corners
   - Hover effect

**Tips Gambar:**
- Resolusi minimum: 1200 x 800 px
- Format: JPG atau PNG
- Compress jika lebih dari 2MB
- Nama file deskriptif

### 5. **Key Highlight yang Powerful**

**âŒ Kurang Menarik:**
```
DRW Foundation memberikan bantuan.
```

**âœ… Powerful:**
```
DRW Foundation menyalurkan 5.000 paket sembako senilai Rp 500 juta 
kepada korban banjir di 5 kecamatan Jakarta Timur, menjangkau lebih 
dari 10.000 jiwa dalam 3 hari.
```

**Formula Key Highlight:**
- Siapa + Apa + Berapa + Dimana + Kapan
- Angka spesifik (5.000 paket, Rp 500 juta)
- Dampak nyata (10.000 jiwa)

### 6. **Keyboard Shortcuts**

- **Ctrl/Cmd + B** = Bold
- **Ctrl/Cmd + I** = Italic
- **Ctrl/Cmd + K** = Insert Link
- **Ctrl/Cmd + S** = Save (di halaman edit)

## ğŸ†˜ Troubleshooting

### Masalah: Editor tidak muncul

**Solusi:**
1. Refresh halaman (F5)
2. Clear browser cache
3. Gunakan browser modern (Chrome, Firefox, Edge)

### Masalah: Template tidak masuk ke editor

**Solusi:**
1. Pastikan Anda di tab yang benar (ID/EN)
2. Klik template sekali lagi
3. Refresh halaman jika perlu

### Masalah: Preview tidak sesuai

**Solusi:**
1. Pastikan sudah isi judul dan excerpt
2. Tunggu beberapa detik untuk render
3. Refresh preview dengan toggle Edit â†’ Preview

### Masalah: Gambar tidak ter-upload

**Solusi:**
1. Cek ukuran file (max 2MB)
2. Cek format file (JPG/PNG)
3. Cek koneksi internet
4. Coba compress gambar terlebih dahulu
